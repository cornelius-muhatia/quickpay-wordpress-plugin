Quickpay Gateway Fast and convenient way to process card payments.

== Description ==

Your site takes the payment information and passes it to the payment processor behind the scenes. 
Users never leave your site. With this method, your site is not handling any of the payment information, you have a limited amount of risk. User never leaves you site, you can completely customize the experience and itâ€™s easier to confirm that a successful payment was made.

== Installation ==

1. Upload plugin to your blog, install and activate plugin
1. Enter merchant details provided by Quickpay
1. Use shortcode [quickpay-checkout/] to include the plugin anywhere in your blog (For example if you want to the user to pay 200 use; [quickpay_checkout]2000[/quickpay_checkout]  for custom amount use [quickpay_checkout/])
